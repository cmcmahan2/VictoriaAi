The Pipeline

Five phases. One command.

Run the full pipeline end-to-end, or drop in at any phase with existing data.

1

Business Discovery

Finds BC businesses with weak or no web presence using Google Maps Places API with Playwright scraping fallback. Each lead is scored 1–10 on web presence weakness and saved to leads.json.

Google Places APIPlaywrightYelp · YellowPages · BBBBC Registryleads.json
2

Business Intelligence Scraping

Builds a complete profile: existing website screenshots & colour palette, Google Business photos and reviews, social media presence, review platform aggregation, and competitor benchmarking.

Website ScreenshotsGMB Photos & ReviewsFacebook · Instagram · LinkedInCompetitor Analysisprofile.json
3

Website Generation

Builds a production-ready site using the right stack for the business type. Static HTML for tradespeople, Next.js + Tailwind for professional services. Includes 5 required pages, SEO baseline, and local schema markup.

Static HTML/CSS/JSNext.js + TailwindJSON-LD SchemaMobile-FirstGA4 · Sitemap · OG Tags5 pages built
4

AI Opportunity Audit PDF

Produces a professional 10-page PDF proposal for the business owner covering 5 AI & automation dimensions, with a 30/60/90 day implementation roadmap. Written for a non-technical audience.

AutomationsCustomer ServiceMarketingOperationsRevenuePDF delivered
5

Auto-Deployment

Pushes the finished site live to a unique Netlify subdomain so the client can see a working preview before any commitment. The live URL is injected into the PDF cover page automatically.

NetlifyAuto URL capturedeployment.jsonSite live
Built For BC

Every detail tuned for British Columbia businesses

From Canadian spelling to GST/PST references — this tool speaks BC.

🏔️

BC-Specific Discovery

Searches target BC cities and regions. Checks the BC Registry for active business status and Facebook Pages for businesses with no website.

📱

Mobile-First Output

Over 70% of local search happens on mobile. Every generated site is built mobile-first with sub-2s load targets and no unnecessary JavaScript.

🔍

Local SEO Baked In

Every site ships with local business JSON-LD schema, city-specific copy, sitemap.xml, Open Graph tags, and a Google Analytics 4 placeholder.

🤖

AI-Powered Copy

Content is generated from real scraped data — real services, real reviews, real phone numbers. Missing fields get clear TODO placeholders, never invented details.

⚡

Right Stack Per Business

Plumber? Static HTML. Dental clinic? Next.js + Tailwind. Retailer? Next.js + Snipcart. The tool picks the right tool for the job automatically.

📄

PDF Proposal Included

Every run produces a branded AI opportunity report — ready to hand to the business owner with the live preview URL on the cover page.

AI Opportunity Audit

Five dimensions. Clear ROI. Written for owners, not engineers.

The PDF report evaluates every business across five areas and maps a 30/60/90 day implementation roadmap.

Automations

Booking, invoicing, follow-ups, and instant quote calculators. High-value if phone present but no online booking.

Customer Service

FAQ chatbot (Claude API), after-hours AI coverage, missed call handling with voicemail-to-email summaries.

Marketing

Email sequences, review generation via SMS, AI content calendar, Google Ads gap analysis, local SEO fixes.

Operations

Job scheduling optimisation, inventory reorder alerts, AI-assisted document generation and cost tracking.

Revenue

Upsell automation, adjacent service opportunities, referral tracking, and transparent pricing page strategy.

How It Works

One command. Five phases. Site live.

1

Provide a city + category

Tell the tool what type of business and which BC city to target.

2

Discovery runs automatically

The tool finds leads, scores their web presence weakness, and ranks them.

3

Profile is built

Every public data source is scraped: website, GMB, social, reviews, competitors.

4

Site + PDF generated

A production website and AI opportunity report are built from the real data.

5

Live in minutes

The site deploys to Netlify. The PDF cover shows the live preview URL.

# Full pipeline — discover, build, audit, deploy
python src/cli.py run --type plumber --city Victoria

# Single known business
python src/cli.py run --name "Okanagan Electric" --address "789 Main St, Kelowna BC"
